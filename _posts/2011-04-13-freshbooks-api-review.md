[<img src="http://kinlane-productions.s3.amazonaws.com/freshbooks-invoice-logo.gif" alt="" align="right" />][1][Freshbooks][1] provides time tracking, expense and invoicing Software as a Service (SaaS).

The [Freshbooks API][2] is a RESTful API, using OAuth for authentication. The API provides sixteen endpoints, providing the same functionality through the API, as whats available through the web application.

The Freshbook API community provides Ruby, Python, PHP, CakePHP, Perl, JavaScript, ActionScript, Java, and .NET sample code libraries.
