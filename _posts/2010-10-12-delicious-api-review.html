--- published: true layout: post title: 'Delicious API Review' image: http://kinlane-productions2.s3.amazonaws.com/api-evangelist/delicious-logo.jpg --- Delicious is a social bookmarking web service for storing, sharing, and discovering Web bookmarks.

Delicious has a RESTful API, with a HTTP-Auth authentication. Using the API, you can read and write Delicious bookmarks and tags. Delicious provides some PHP Curl scripts for connecting, but no other language kits.
