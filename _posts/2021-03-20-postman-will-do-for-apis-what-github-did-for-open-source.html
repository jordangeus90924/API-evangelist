That is how I see Postman. GitHub changed my relationship with code in many ways. Github plays a significant role in how I produce, manage, and evolve the code that I produce for my own side projects and businesses, as well as the code I engage with as part of my job and the projects I work on. First, I published my thoughts about what GitHub did for code, and now I want to explore what Postman is doing for APIs.
