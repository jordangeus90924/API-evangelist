Trends

When you spend a lot of time looking at the API space you start seeing new patterns of how APIs are being used in new and exciting ways.

Its not the early days of API mashups, there are numerous new ways that API usage is evolving--further connecting the data and resources we need to build the next generation of web and mobile apps.
Aggregation
Reciprocity
Real-Time
Voice
Single Page Apps (SPA)

These are just a handful of the API trends I'm watching.  I'll be rolling out other sections for trends in the future.  If you see a trend in the space you think I should be focusing on, please let me know.
