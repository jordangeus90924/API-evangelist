--- layout: post title: 'Stack Oververflow API Review' url: 'http://apievangelist.com2011/01/20/stack-oververflow-api-review/' image: '' --- StackApps is an API for the family of Q&A websites using the Stack Exchange engine including Stack Overflow, Server Fault and Super User. StackApps provides a RESTFul API using a basic application key authentication. The API provides access to Stack Apps user data as well as questions, answers and comments. No code samples or language packs are provided, they provide a link to the World's Worst Stack Overflow Clone which is a sample application written in C and a hello world URL example.
