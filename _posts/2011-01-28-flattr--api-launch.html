--- published: true layout: post title: 'Flattr - API Launch' image: http://kinlane-productions2.s3.amazonaws.com/flattr_logo.jpg --- Flattr the social micro-payment site, made their API public today.

The API comes in two flavors.

  • Javascript API
  • REST API
Flattr provides application registration and authentication with oAuth.

They also have a Flattr developers forum to support the community.
